bool MessageDecoder::compare ( const char *  str)

Check to see if a string matches the message key.

Note
For example, if this message decoder was meant to decode a "ACTIONMESSAGE" and the key of an action message was "action-message", then a '\0'-terminated string that represents the action key will return true, and all others will return false.
