Detailed Description

template<typename T, unsigned int BufferSize>
class ledstd::RingBuffer< T, BufferSize >

This class represents a queue that uses a circular buffer for maintaining the elts.

Note
Do not overfill buffer. If you push more and BufferSize elts into the RingBuffer, then there will be undfined behavior.
